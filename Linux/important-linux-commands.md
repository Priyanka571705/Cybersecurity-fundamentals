# Important Linux Commands for SOC Analysts

## Introduction
Linux commands are essential for SOC analysts to navigate systems
and analyze logs.

## ls 
Used to quickly check log files, scripts, or suspicious files in a directory.
## pwd
Helps analysts understand their current location in the file system.
## cd /var/log
Used to move into log directories for investigation.
## cat auth.log
Used to quickly view authentication logs.
## chmod 644 filename
Helps ensure sensitive files are not writable by unauthorized users.

## ls -l
Used to detect misconfigured permissions.
## whoami
Helps verify the privilege level during investigations.
## uname -a
Used to identify system details during incident analysis.
